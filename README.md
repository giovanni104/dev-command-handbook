# Dev Command Handbook

Repositorio personal de referencia rápida con comandos, configuraciones y apuntes técnicos en formato Markdown sobre las tecnologías que uso en mi día a día como desarrollador.

El objetivo de este repositorio es centralizar comandos útiles para consultar rápidamente durante el desarrollo de proyectos backend, frontend, bases de datos, control de versiones y herramientas de entorno.

## Convención de nombres

Los archivos se nombran usando minúsculas, guiones medios y una descripción clara del contenido.

Ejemplos:

```bash
guia-comandos-git.md
guia-comandos-docker.md
guia-comandos-spring-boot.md
guia-comandos-mysql.md
```

## Uso del repositorio

Cada archivo contiene comandos organizados por secciones, acompañados de una breve explicación y ejemplos prácticos.

Este repositorio está pensado como una guía de consulta rápida para estudiar, repasar y aplicar comandos en proyectos reales.
